# uranai-maker — オリジナル占いアプリ ジェネレーター

ユーザーへの質問に答えてもらうだけで、オリジナルの占いアプリ（HTML/CSS/JS）を生成する。
生成物は Cloudflare Pages にそのままアップできるフォルダ構成で出力する。

---

## ステップ1: インタビュー

以下の質問を**1つずつ**順番に聞いていく。全部まとめて聞かない。
前の答えを受けて次の質問をするように、会話を自然につなぐ。

### 質問リスト

1. **テーマ**
   「どんな占いにしますか？（例：昆虫占い、お魚占い、家電占い、なんでもOK）」

2. **キャラクター数**
   「何種類のキャラクターに分けますか？（例：6・8・12など）」

3. **キャラクター名**
   「キャラクターの名前を教えてください。{数}種類分。
   （思いつかない場合は『考えて』と言ってもらえれば提案します）」

4. **各キャラクターの説明**
   「各キャラクターの性格・特徴を一言で教えてください。
   （思いつかない場合は『考えて』と言ってもらえれば提案します）」

5. **判定方法**
   「判定方法はどうしますか？
   　A) 誕生日（生年月日）を入力して判定
   　B) 質問に答えて判定（性格診断タイプ）
   　C) 両方使う」

   - **A または C を選んだ場合**: 誕生日から判定するアルゴリズムを自動設計する（ユーザーへの説明は不要）
   - **B または C を選んだ場合**: 「どんな質問をしますか？いくつくらい？（例：3〜5問）」と聞く。質問内容は提案してもよい。

6. **デザインの雰囲気**
   「デザインはどんな雰囲気にしますか？
   　例：かわいい系・クール系・ポップ系・和風・シンプル など」

7. **カラー**
   「メインカラーはありますか？（例：ピンク、青、緑など。なければ『おまかせ』でOK）」

8. **タイトル**
   「占いアプリのタイトルを教えてください。（例：『あなたはどの昆虫？』）」

## ステップ2: 確認

インタビューが終わったら、収集した情報を箇条書きで整理してユーザーに見せる。
「この内容でアプリを作ります。よろしいですか？」と確認してから生成に進む。

## ステップ3: アプリ生成

### 出力フォルダ構成

```
{テーマ名}-uranai/
└── index.html   （CSS・JS すべてこの1ファイルに含める）
```

### index.html の要件

- **1ファイル完結**（外部ライブラリ不要、CDN も使わない）
- HTML / CSS / JavaScript をすべて index.html に書く
- スマートフォンでも見やすいレスポンシブデザイン
- 文字コードは UTF-8

### 画面構成

1. **タイトル画面**
   - タイトルとキャッチコピーを表示
   - 「診断スタート」ボタン

2. **入力・質問画面**
   - 判定方法 A: 生年月日の入力フォーム（年・月・日）
   - 判定方法 B: 質問を1問ずつ表示（選択肢ボタン）
   - 判定方法 C: 誕生日入力 → 質問の順に進む

3. **結果画面**
   - キャラクター名を大きく表示
   - キャラクターの説明文
   - 大きな絵文字でキャラクターを表現（画像不使用）
   - 「もう一度診断する」ボタン

### 判定アルゴリズム

**誕生日判定の場合**:
生年月日の数字を使ってキャラクター数で割り算し、余りでキャラクターを決定する。
全キャラクターに偏りなく振り分けられるよう工夫する。

**質問判定の場合**:
各選択肢に点数を割り当て、合計点でキャラクターを決定する。

**両方の場合**:
誕生日スコアと質問スコアを組み合わせて最終判定する。

### デザイン実装

ユーザーが指定した雰囲気・カラーをもとに CSS を作成する。
絵文字をキャラクターアイコンとして活用する（例：昆虫なら🐛🦋🐝🐞など）。

## ステップ4: 完成後のメッセージ

アプリ生成後、以下を伝える：

```
✅ 占いアプリが完成しました！

📁 フォルダ: {テーマ名}-uranai/

【ローカルで確認する方法】
  open {テーマ名}-uranai/index.html

【Cloudflare Pages で公開する方法】
1. https://dash.cloudflare.com にアクセス
2. Workers & Pages → アプリケーションを作成する
3. 「Pages を導入しようとお考えですか？始める」をクリック
4. ファイルをドラッグアンドドロップ → 始める
5. プロジェクト名を入力 → プロジェクトの作成
6. {テーマ名}-uranai フォルダをアップロード
7. サイトをデプロイ → 完成！
```
